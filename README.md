# StudySync

AI-Powered Student Productivity and Collaborative Learning Platform

## Features

- User Authentication
- Notes Management
- Task Management
- AI Assistant (Groq)
- AI Quiz Generator
- Progress Tracking
- Dashboard Analytics

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript
- Groq API

## Setup

1. Clone repository
2. Install requirements
3. Create MySQL database
4. Configure API key
5. Run app.py
